A lightweight, high-\/performance HTTP web framework written in C, inspired by Gin (Golang) and built on top of libuv, llhttp, and c\+JSON.\hypertarget{md_README_autotoc_md16}{}\doxysection{\texorpdfstring{Features}{Features}}\label{md_README_autotoc_md16}

\begin{DoxyItemize}
\item 🚀 High performance using libuv for asynchronous I/\+O
\item 🔧 Middleware support (logger, recovery, auth, static files)
\item 🌐 RESTful API routing with parameter handling
\item 📦 JSON support via c\+JSON
\item 🔍 URL parsing and query string handling
\item ⚡ Keep-\/alive connection support
\item 🛡️ Graceful error handling
\end{DoxyItemize}\hypertarget{md_README_autotoc_md17}{}\doxysection{\texorpdfstring{Dependencies}{Dependencies}}\label{md_README_autotoc_md17}

\begin{DoxyItemize}
\item \href{https://github.com/libuv/libuv}{\texttt{libuv}} -\/ Asynchronous I/\+O library
\item \href{https://github.com/nodejs/llhttp}{\texttt{llhttp}} -\/ HTTP parser
\item \href{https://github.com/DaveGamble/cJSON}{\texttt{c\+JSON}} -\/ JSON parser
\end{DoxyItemize}

These dependencies are automatically fetched during build via CMake\textquotesingle{}s Fetch\+Content.\hypertarget{md_README_autotoc_md18}{}\doxysection{\texorpdfstring{Building}{Building}}\label{md_README_autotoc_md18}
